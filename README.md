# VoronoiLinker
Blender Addon for Node Editor which adds three new awesome features:
* Voronoi Linker  
* Voronoi Preview  
* Voronoi Mixer  
* Voronoi QuickMath  
* Voronoi Swaper  
* Voronoi Hider  
* Voronoi MassLinker  
* Voronoi PropSelector

See wiki for details https://github.com/ugorek000/VoronoiLinker/wiki
